基本の出力

Pythonでは print() を使って画面に出力できます。

print("Hello, World!")

変数とデータ型

Pythonの基本的なデータ型には、整数、浮動小数点数、文字列などがあります。

x = 10
pi = 3.14
message = "こんにちは、Python!"
print(x, pi, message)

条件分岐

if 文を使うことで条件に応じた処理ができます。

number = 7
if number % 2 == 0:
    print("偶数です")
else:
    print("奇数です")

ループ処理

for や while を使って繰り返し処理ができます。

for i in range(5):
    print("ループ", i)
